## About

- [CS50 Python](https://cs50.harvard.edu/python/) is a Harvard University's introductory course to Programming with Python. It covers topics such as functions, conditionals, loops, data structures, file I/O, regular expressions, unit tests, and object-oriented programming.

- The course's content is divided in 10 weeks, each composed of a Lecture and a Problem Set.

- This repository contains my solutions for the Problem Sets, organized by week. The [Final Project](https://cs50.harvard.edu/python/project/) is in a separate repository.
