## 🚀 Projects and AI Concepts

Below is a summary of the 12 comprehensive projects completed, categorized by the core AI topic they cover:

### 1. Search
* **[Degrees](degrees/)**: Finds the shortest path between two actors using Breadth-First Search (BFS) to determine their "degrees of separation."
* **[Tic-Tac-Toe](tictactoe/)**: Implements an unbeatable AI that plays Tic-Tac-Toe optimally using the **Minimax** algorithm (with Alpha-Beta Pruning).

### 2. Knowledge
* **[Knights](knights/)**: Solves Knights and Knaves logic puzzles using **Propositional Logic** and a model-checking algorithm.
* **[Minesweeper](minesweeper/)**: An AI that plays Minesweeper by maintaining a **Knowledge Base** and making safe inferences using subset tracking.

### 3. Uncertainty
* **[Pagerank](pagerank/)**: Ranks web pages by importance using both a random surfer model and the iterative **Markov Chain** algorithm.
* **[Heredity](heredity/)**: Calculates the probability of individuals having a certain genetic trait using **Bayesian Networks** and joint probabilities.

### 4. Optimization
* **[Crossword](crossword/)**: Generates complete crossword puzzles by treating the board as a **Constraint Satisfaction Problem (CSP)** and employing node/arc consistency and backtracking search.

### 5. Learning
* **[Shopping](shopping/)**: Predicts whether online shopping customers will complete a purchase based on session data using a **k-Nearest Neighbors (k-NN)** classifier.
* **[Nim](nim/)**: Trains an AI to play the game of Nim flawlessly against human opponents using **Q-Learning** (Reinforcement Learning).

### 6. Neural Networks
* **[Traffic](traffic/)**: Identifies which traffic sign appears in a photograph by building and training a **Convolutional Neural Network (CNN)** using TensorFlow.

### 7. Language
* **[Parser](parser/)**: Parses English sentences to determine their structure and extract noun phrases using **Context-Free Grammars (CFG)** and the `nltk` library.
* **[Attention](attention/)**: Analyzes and visualizes the self-attention mechanisms of the **BERT Transformer model** when predicting masked words in a sentence.

## 🛠️ Technologies Used
* **Python 3**: The core programming language used for all algorithm implementations.
* **TensorFlow & Keras**: Used for building deep neural networks and training the CNN.
* **Hugging Face Transformers**: Leveraged for working with large language models like BERT.
* **Natural Language Toolkit (NLTK)**: Used for sentence tokenization and grammatical parsing.

## 📥 Usage
To run any of the projects locally, simply navigate to the project directory and install the required dependencies (if any are listed in `requirements.txt`). Feel free to explore the code to understand the underlying AI implementations!
